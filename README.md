# foundation_robot
This is a foundation monitor project

Environment need:
```
requests
```
- Maybe you need run:
```
python -m pip install requests
python -m pip install --upgrade pip
```

Change your own foundation code in `fund_codes.txt` line by line

Input `python3 main.py` to run it.
